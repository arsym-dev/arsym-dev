# Reverse Engineering & Fan Patches

### Remember11: The Age of Infinity (PC, PS2)
- [Remember11-Script-Editor](https://github.com/arsym-dev/Remember11-Script-Editor) — Rich GUI for arbitrary modifications to the proprietary scenario format; compares Japanese and translated text side-by-side; translation sanity-check tools
- [Remember11-RPPA-Notes](https://github.com/arsym-dev/Remember11-RPPA-Notes) — Notes documenting the engine and file formats (Raku)
- Used to create the [Gestalt Edition fan patch](https://old.reddit.com/r/InfinitySeries/comments/mbk11z/remember11_gestalt_edition_repost/)

### Ever17: The Out of Infinity (PC, PS2)
- [Ever17-Exe-Modifier](https://github.com/arsym-dev/Ever17-Exe-Modifier) — Edit fonts, increase skip speed, prevent music from restarting
- [Ever17-Scenario-Editor](https://github.com/arsym-dev/Ever17-Scenario-Editor) — Modify the proprietary scenario format
- [Ever17-CPS-Converter](https://github.com/arsym-dev/Ever17-CPS-Converter) — Convert and edit images (CPS format)
- Used to create the [Himmel Edition fan patch](https://old.reddit.com/r/InfinitySeries/comments/mbkbhn/ever17_himmel_edition_repost/)

### 12Riven (PC)
- [Repository](https://github.com/Exormeter/12Riven_ScriptEditor) (lead: Exormeter)
- Assisted in reverse engineering commands and adapting the Remember11 script editor to accommodate 12Riven

### Endonesia (PS2)
- [endonesia-tool](https://github.com/arsym-dev/endonesia-tool) — Fan translation project; tools for editing images, animations, and text

### 428: Shibuya Scramble (PC)
- [428-Text-Speed-Patch](https://github.com/arsym-dev/428-Text-Speed-Patch) — Change the rate at which text is displayed

### Vampirdzhija Vjedogonia (PC)
- [vjedogonia-tools](https://github.com/arsym-dev/vjedogonia-tools) — Tools for editing images and text
